## COVID-19 Live Dashboard とは？

新型コロナウイルス感染症（COVID-19）の累計人数を、グラフで表示する Web アプリケーションです。<br />
日本国内と世界各国、それぞれの累計を確認できます。

COVID-19 Live Dashboard<br />
https://covid-19-live-dashboard-38k.web.app/
